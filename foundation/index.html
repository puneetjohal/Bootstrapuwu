Team uWu's Foundation Demo

Welcome to our Foundation demo! Here we will demonstrate some of the usefule capabilities of the Foundation for Sites front-end framework. The easiest way to test out these features for yourself is to copy the following CDN link into the head of your HTML file:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.0/dist/css/foundation.min.css">

Key Features of Foundation

XY Grid

The XY Grid allows for better vertical and horizontal alignment of objects. The following is an example of the XY Grid:

Each row contains a maximum of 12 cells. Extra cells will overflow onto the next row. Gutters can be added by setting grid-margin-x or grid-padding-x. You can also control the size of cells and how many cells will display on each row on small, medium, or large screens. Resize your browser window (CTRL + or CTRL -) to see this effect on the grid below.
